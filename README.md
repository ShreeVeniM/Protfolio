# Shree Veni Muralitharan — Portfolio

Personal portfolio site — Data Analyst & BI Developer.

## Structure
- `index.html` — page markup
- `styles.css` — all styling
- `script.js` — cursor glow, scroll fade-in, project filters, resume button handlers
- `Profile.jpg` — profile photo (add yours)
- `Resume.pdf` — downloadable resume (add yours)

## Deploy
Static site, no build step. Deploy directly on [Vercel](https://vercel.com) — import this repo and use framework preset "Other".
